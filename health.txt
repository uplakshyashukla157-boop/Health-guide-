    <h2>Fever</h2>
    <p><b>Symptoms:</b> High temperature, chills, weakness.</p>
    <p><b>Home Care:</b> Rest, drink water, eat light food.</p>
    <p class="warning">See a doctor if fever is very high or lasts several days.</p>
</section>

<section class="health-section" id="cold">
    <h2>Cold & Cough</h2>
    <p><b>Symptoms:</b> Runny nose, sneezing, cough.</p>
    <p><b>Home Care:</b> Rest, warm fluids, hydration.</p>
    <p class="warning">Seek medical help if breathing becomes difficult.</p>
</section>

<section class="health-section" id="headache">
    <h2>Headache</h2>
    <p><b>Symptoms:</b> Head pain, tiredness, sensitivity to light.</p>
    <p><b>Home Care:</b> Rest, water, proper sleep.</p>
    <p class="warning">See a doctor for sudden severe headaches.</p>
</section>

<section class="health-section" id="stomach">
    <h2>Stomach Pain</h2>
    <p><b>Symptoms:</b> Pain, bloating, nausea.</p>
    <p><b>Home Care:</b> Fluids, bland food, avoid spicy foods.</p>
    <p class="warning">See a doctor for severe pain or persistent vomiting.</p>
</section>

<section class="health-section" id="hand">
    <h2>Hand Pain</h2>
    <p><b>Symptoms:</b> Pain, stiffness, swelling.</p>
    <p><b>Home Care:</b> Rest the hand and avoid heavy lifting.</p>
    <p class="warning">See a doctor if pain follows an injury or causes numbness.</p>
</section>

<section class="health-section" id="leg">
    <h2>Leg Pain</h2>
    <p><b>Symptoms:</b> Pain, cramps, swelling.</p>
    <p><b>Home Care:</b> Rest, gentle stretching, hydration.</p>
    <p class="warning">Seek medical help for severe swelling or sudden pain.</p>
</section>

<script>
function searchContent() {
    let input = document.getElementById("searchInput").value.toLowerCase();
    let sections = document.getElementsByClassName("health-section");

    for(let i = 0; i < sections.length; i++) {
        let text = sections[i].innerText.toLowerCase();

        if(text.includes(input)) {
            sections[i].style.display = "block";
        } else {
            sections[i].style.display = "none";
        }
    }
}

function findHospital() {
    document.getElementById("result").innerHTML =
        "Opening nearby hospitals in Google Maps...";

    window.open(
        "https://www.google.com/maps/search/hospitals+near+me",
        "_blank"
    );
}
